Program Specifications

This is Real Time Battleship Game. Enemy battleship attack to player's battleship with missile by the random interval time. Player's battleship has defense missile, close-in weapon system and the physical strength of battleship. If player do not sink all enemy in battlefield, player's battleship is destroyed because defense weapon has limit of numbers.

  • Real Time Battleship Game - Prayer vs. Computer.
  • Include interactive events: Launching an flighting of missile, Tracing tragted misile in rader, Defense system reaction, Destroying missiles
  • Accepts input from the user using different mechanisms: Object moving, Tracing the moving coordinate, - Auto reaction for object changes in the element rang, Event process for image
  • Store data in the browser for reuse between sessions
  • Queries oneline web API resource for data: Request weather information and display this information in CIC
  • Performs validation on user input

I used the canvas tag first time to display Defense Radar.
