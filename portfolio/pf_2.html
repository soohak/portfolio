Javascript Client Side Programming Project

This project is for Client-side Javascript programming to understand Fundamentals DOM with HTML and CSS.

Program Specifications

Create a dynamic HTML5 web page to displays Game which User can set the game board size. For user's mouse click in game board, the application will reflect the sound and count score for result about zombie shooting in screen.

  • Zap-A-Zombi: Hell Gate
  • User can set the row and column to make game board(size)
  • Re-set button can initialize the setting
  • User can shoot the zombie in game board by mouse click.
  • Various sound effects for the game events: shotgun sound for mouse click, zombie sound and backgroud music.
  • If mouse click timing is good, score is counted.

Sound reaction for the mouse click is slow. In my case, I used the touch screen.
When I had made this game, My understanding was not enough about Javascript DOM. So, I don't think this code is nice. However, I like this poor code because I did my best to make this with just my logic without enough understanding for DOM.
